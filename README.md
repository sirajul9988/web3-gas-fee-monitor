# Web3 Gas Fee Monitor

This is a simple Web3 project.  
It helps you track gas fees on a blockchain network.

This project is made for beginners who want to learn how transaction fees work.

## What This Project Does
- Check current gas fee
- Compare with last gas fee
- Show status (high or low)
- Save gas history to file

## Tools Used
- Node.js
- Simple JavaScript
- Public gas API (mock)
- Local JSON file for storage

## How To Install
1. Install Node.js
2. Download this repository
3. Open terminal in project folder
4. Run:
   npm install
5. Start the app:
   node app.js

## Folder Structure
- app.js = main file
- config.js = project settings
- gas.js = gas logic
- api.js = API calls
- logger.js = save logs
- data/ = gas history

## Example Use
- App checks gas price
- App prints fee status
- App saves result to file
